# Screenshots

Place your demo screenshots here:

1. **terminal_ui.png** - Screenshot of the main Streamlit UI showing:
   - Bid/Ask prices
   - Historical chart
   - Volume chart
   - Statistics

2. **historical_chart.png** - Generated automatically when running `historical_viewer.py`

## How to Take Screenshots

### For the Streamlit UI:
1. Run: `streamlit run src/realtime_ui.py`
2. Enter a ticker symbol (e.g., AAPL)
3. Wait for data to load
4. Take a full-page screenshot
5. Save as `terminal_ui.png` in this directory

### For Historical Charts:
The historical viewer automatically generates charts:
- Run: `cd src && python historical_viewer.py`
- Charts are saved automatically to the project root
- Move them here for organization

## Screenshot Tips

- Use a clean, light background
- Ensure text is readable
- Show the full interface
- Include real data (not loading states)
- Capture during market hours for live data
